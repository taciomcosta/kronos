## Installation
### Linux

**Debian/Ubuntu**
```
sudo apt-get install kronos
```

**Arch**
```
sudo pacman -S kronos
```

### macOs
```
$ brew tap taciomcosta/kronos
$ brew install kronos
```

### Windows
```
choco install kronos
```
